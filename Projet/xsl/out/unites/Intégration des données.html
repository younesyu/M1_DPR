Résumé :

L’objectif est d’intégrer des données provenant de plusieurs sources de données hétérogènes, pour une exploitation unifiée de façon matérialisée ou virtuelle. Y seront abordés les concepts et architectures d’entrepôts de données et de médiateurs, ainsi que le traitement optimisé de requêtes. Contenu  : Mise en niveau en base de données Entrepôt de données Médiation Optimisation (SQL3)
