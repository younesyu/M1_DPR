Cloud computing

crédits : 3
résumé : Le Cloud Computing désigne un univers de ressources (stockage, calcul) utilisables de manière élastique (à la demande) à travers Internet. Dans cette UE, nous étudierons les principaux concepts : virtualisation, sécurisation, les modèles de services IaaS, PaaS, SaaS, les types de cloud (privé, public, etc.). Nous examinerons aussi les défis, risques et opportunités soulevés par l’adoption du Cloud. Les travaux pratiques seront réalisés sur une des plateformes de Cloud comme Microsoft Azure ou AWS.
